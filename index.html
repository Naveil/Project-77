My Second Console Project

In order to set up Visual Studio Code, you must create a new folder and name it whatever you desire. Then, open that folder on Visual Studio Code. Last, add the files index.html and main.js.



The console screen is accessed by right clicking on the screen, selecting inspect, then going to the console. It is used to see how your code works, and you are able to print text on the console by creating a variable, then using console.log() with the variable name in the brackets.
